## ✨ Overview
**Wind V Client** は、Minecraft 1.8.9のPvP体験を次のレベルへ引き上げるために開発された、軽量・高機能なユーティリティクライアントです。  
パフォーマンス、カスタマイズ性、そして「使い心地」に徹底的にこだわりました。

---

## 🚀 Key Features

### 🇯🇵 Japanese IME Patch
Minecraft 1.8.9で長年問題だった「日本語入力」のバグを解消。チャットや看板、本で快適に日本語を入力できます。

### 🎨 Modern HUD Editor
すべてのHUD要素をドラッグ＆ドロップで配置可能。スナップ機能により、ピクセル単位の美しいレイアウトを簡単に作成できます。

### ⚔️ 1.7 Animations
1.8.9上で、1.7風の剣のガード、アイテムの使用、殴りモーションを忠実に再現。PvPの打撃感とリズムを向上させます。

### ⚡ Extreme Performance
Entity Culling、メモリ最適化、そして洗練された内部設計により、低スペックPCからハイエンドPCまで、最大限のFPSを引き出します。

## 🛠️ Included Mods
Wind V Clientには、以下のModが標準搭載されています（すべて設定可能）：

*   **Combat**: Keystrokes, CPS Display, Toggle Sprint, Reach Display
*   **Utility**: Armor Status, Potion Status, Direction HUD (Compass), Zoom
*   **Visual**: Custom Crosshair Editor, Motion Blur, Fullbright, Time Changer
*   **System**: FPS Counter, Ping Display, Memory Usage, Pack Display

## 📥 Installation

1. [Modrinth](https://modrinth.com/mod/windv-client) から最新の `.jar` ファイルをダウンロードします。
2. Minecraft Forge 1.8.9 がインストールされていることを確認します。
3. ダウンロードしたファイルを `%appdata%/.minecraft/mods` フォルダに移動します。
4. Minecraftを起動し、設定メニューから Wind V Client の機能を有効にしてください。
