**作成日時：** 2026-05-01 19:58

## 概要
Minecraft 1.8.9 Forge向けPvP特化軽量クライアント「Wind V Client (WVC)」のMVP実装。

## 実装済みコンポーネント

### プロジェクト構成
```
WindVClient/
├── build.gradle               # Forge 1.8.9 ビルド設定
├── settings.gradle            # Gradleプロジェクト設定
├── gradlew.bat                # Gradle Wrapper (Windows)
├── gradle/wrapper/
│   └── gradle-wrapper.properties
├── src/main/
│   ├── java/dev/windv/wvc/
│   │   ├── WVCMod.java               # メインModクラス (@Mod)
│   │   ├── config/
│   │   │   └── WVCConfig.java        # 設定管理（Forge Config API）
│   │   ├── module/
│   │   │   ├── WVCModule.java        # モジュール基底クラス
│   │   │   ├── ModuleManager.java    # モジュール登録・管理
│   │   │   └── hud/
│   │   │       ├── FpsModule.java    # FPS表示
│   │   │       ├── CpsModule.java    # CPS表示（循環バッファ方式）
│   │   │       └── PingModule.java   # Ping表示（1秒キャッシュ）
│   │   └── render/
│   │       └── HudRenderer.java      # HUD描画エンジン（サイバー系）
│   └── resources/
│       ├── mcmod.info
│       └── META-INF/MANIFEST.MF
└── assets/
    └── logo.png
```

### 実装したモジュール
| モジュール | 説明 | 軽量化対策 |
|-----------|------|-----------|
| FPS | Minecraft.getDebugFPS()で取得 | 毎フレームAPIコール1回のみ |
| CPS | 循環バッファで直近1秒計測 | アロケーションなし |
| Ping | NetworkPlayerInfo API | 1秒キャッシュで頻繁呼び出し回避 |

### HUDデザイン
- カラーパレット：深紺背景 + 青紫ボーダー + シアン/黄/赤の値テキスト
- 半透明パネル：GL11直接描画（テクスチャなし）
- 値に応じた色変化：FPS 60↑=シアン / 30↑=黄 / 以下=赤

## ビルド手順
1. JDK 8 + Gradleインストール確認
2. `gradlew setupDecompWorkspace`
3. `gradlew build`
4. `build/libs/WindVClient-1.0.0.jar` を `.minecraft/mods/` に配置

## トラブルシューティング（2026-05-01）
### ビルド失敗の分析
- **現象:** Gradleビルドでエラーまたは多数の警告が発生。
- **原因:** 
    1. **Javaバージョン不一致:** 現在の環境が Java 26 であるが、Forge 1.8.9 は **Java 8** 必須。
    2. **Gradle構文:** 新しいGradle環境で古いGroovy DSL構文に対する警告が発生。
